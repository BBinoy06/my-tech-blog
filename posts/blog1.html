Why and how I made this journal and what I learnt...

Why I made this journal...

I made this tech journal to track my learning — not just what I learn, but how I feel about it. I started computer science without any background. I didn’t even know what HTML or GitHub was a few months ago.

This journal gives me space to reflect, experiment, and build something that’s mine. It’s also helping me improve my front-end skills by actually building and designing something that grows with me.

What I’ve Learnt So Far:

How to write basic HTML and structure a page

Linking external CSS and fonts using Google Fonts

Hosting a website using GitHub Pages

Creating and styling a sidebar

Making a JavaScript tracker that responds to clicks using localstorage()

Reflection on Building the Site

Building the site helped reinforce how HTML, CSS, and JavaScript work together. I learned how to break down a page into sections and structure the layout cleanly.

Styling was a challenge at first, especially making the sidebar and content boxes look right — but it taught me how to use CSS classes more effectively.

I also had to troubleshoot small issues, like linking files correctly and making sure my GitHub Pages deployment updated properly.

Using localStorage() in JavaScript was a turning point, it showed me how front-end features can store data and make a site feel more interactive.

Next Goals

Keep journaling weekly to track both skills and feelings

Start learning basic back-end concepts (e.g. what servers and databases are)

Try setting up a simple back-end using Node.js and Express
